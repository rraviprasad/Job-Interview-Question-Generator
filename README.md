Designed and developed a chatbot that dynamically generates customized interview questions based on the user's selected domain (e.g., Web Development, Data Science, Networking), helping students and job seekers enhance their interview preparation. Integrated Google Gemini API for generating domain-specific questions, and a responsive front-end using HTML, CSS, and JavaScript to ensure a seamless user experience.
